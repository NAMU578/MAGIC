## 연구 배경

점자는 시각장애인이 글을 읽는 핵심 수단이지만, 점자를 읽을 수 있는 시각장애인의 비율은 제한적이며 점자 자료 자체도 부족합니다. 또한 점자를 모르는 중도 실명자나, 점자 학습 중인 사람에게는 점자 문서에 접근하는 것 자체가 큰 장벽입니다.

**MAGIC 팀**은 이 문제를 해결하기 위해, 손가락에 착용하고 점자 위를 훑기만 하면 그 내용을 음성으로 들려주는 **웨어러블 점자 인식 기기**를 연구합니다.

## 프로젝트 목표

손가락에 착용 가능한 소형 웨어러블 형태로, 점자를 센서로 인식하고 → 블루투스로 전송 → AI가 글자로 해석 → 음성으로 출력하는 전 과정을 구현하는 것이 목표입니다.

- **착용성**: 손가락에 끼우는 골무형 또는 밴드형의 작은 기기
- **자립 동작**: 배터리 내장, 무선(블루투스) 연결
- **정확한 인식**: 광섬유 또는 압력 센서로 점자 돌기 감지
- **지능형 해석**: AI(LLM)로 점자 패턴을 한글로 교정·변환

## 동작 원리

```
점자 돌기 → [센서: 광섬유 또는 압력] → 전기 신호
         → [ESP32: 신호 디코딩 + 블루투스 전송]
         → [스마트폰 앱: AI 점자 해석 + 음성 출력]
```

1. **센서 입력**: 손끝의 센서가 점자 돌기의 형태(6점 패턴)를 감지합니다. 광섬유 방식은 돌기가 광섬유를 눌러 생기는 광량 변화를, 압력 방식은 저항 변화를 측정합니다.
2. **신호 처리·전송**: ESP32 마이크로컨트롤러가 센서 값을 읽어 점자 셀 패턴(6비트)으로 변환하고 블루투스로 스마트폰에 전송합니다.
3. **AI 해석**: 스마트폰 앱이 6비트 패턴을 한글 점자 규칙에 따라 글자로 변환하고, AI가 약자·문맥을 교정합니다.
4. **음성 출력**: 변환된 글자를 TTS(음성 합성)로 읽어줍니다.

## 두 가지 센서 방식

본 프로젝트는 두 가지 센서 방식을 모두 설계해 비교 검증합니다. 자세한 회로·설계는 [하드웨어 설계](hardware) 페이지를 참고하세요.

<div class="table-wrap" markdown="1">

| | 광섬유(POF) 방식 | 압력(FSR) 방식 |
|---|---|---|
| 원리 | 점자 돌기가 광섬유를 눌러 광량 감소 | 점자 돌기 압력으로 저항 감소 |
| 장점 | 차별성, 민감도 | 회로 단순, 디버깅 쉬움 |
| 용도 | 최종 시연 핵심 | 초기 개념 검증·백업 |

</div>
