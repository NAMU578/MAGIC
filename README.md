# MAGIC — 점자 인식 웨어러블 연구 프로젝트

경북과학고등학교 **의공학마법사(MAGIC)** 팀의 연구 홈페이지 저장소입니다.

🌐 **홈페이지**: https://namu578.github.io/MAGIC/

## 이 저장소는?

손가락에 착용하는 점자 인식 웨어러블 기기를 연구하는 팀의 자료실이자 연구 홈페이지입니다. 광섬유·압력 센서로 점자를 읽고, AI가 글자로 해석해 음성으로 들려주는 기기를 만듭니다.

## 홈페이지 수정하기

이 사이트는 GitHub Pages로 운영됩니다. **저장소의 `.md` 파일을 수정하면 홈페이지가 자동으로 갱신됩니다.** 코딩 지식은 필요 없습니다.

자세한 방법은 [홈페이지 수정 방법(CONTRIBUTING.md)](CONTRIBUTING.md)을 참고하세요.

## 폴더 구조

```
index.md              홈 화면
introduction.md       프로젝트 소개
hardware/             하드웨어 설계 (회로·PCB·3D·펌웨어)
timeline.md           연구 일정
members.md            팀 소개
downloads.md          자료실
experiments/          실험 기록
meetings/             회의록
ideas/                아이디어
references/           참고문헌
templates/            문서 양식
assets/               디자인(CSS)·이미지·파일  ← 수정 주의
_layouts/             페이지 틀  ← 수정 주의
```
